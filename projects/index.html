SOPH-SAP-A2182-水性密著剂

磷酸酯高分子

廣泛用於底漆的金屬底材密著促進劑，增加對面塗的密著。

SOPH-SAP-2171-密著剂

磷酸酯高分子

廣泛用於底漆的金屬底材密著促進劑，增加對面塗的密著。

SOPH-SAP-2010-密著剂

特殊含酸高分子

用於烤漆的溶劑型密著劑，促進塗膜與金屬基材的密著，尤其是非鐵金屬。

SOPH-2081-密著剂

有胺基的矽烷

廣泛用於玻璃及金屬底材的前處劑，增加對面塗的密著。

SOPH-SAP-2093-密著剂

環氧基的矽烷混合物

含有環氧基的矽烷化合物，能提升塗膜在玻璃及金屬底材的密著性。

SIVEL-5525-流平剂

丙烯酸共聚物

針對溶劑型設計的非矽酮流平劑，兼具脫泡效果，適合於透明及色漆系統。

SIVEL-5522-流平剂

氟改質高分子

適合於透明及色漆系統，相容性十分優秀，提供優異流平性。

SIVEL-5521-溶剂型流平剂

丙烯酸酯共聚物

適合於透明及色漆系統，相容性十分優 秀，不會造成塗膜混濁與霧影。

SIVEL-5146-平滑剂

改質聚矽氧烷

應用於溶劑型塗料及油墨、UV硬化塗料及油墨。

SIVEL-5141-平滑剂

改質聚矽氧烷

應用於溶劑型塗料及油墨、UV硬化塗料及油墨。

SIVEL-5120-平滑剂

聚酯改質聚矽氧烷

應用於烤漆系統，相容性佳具有耐高溫、重塗姓的特色乾爽手感。

SIVEL-5102-平滑剂

改質聚矽氧烷

適用於溶劑型的矽酮平坦、平滑劑，具有乾爽的平滑觸感。

ELIFOM-A6320-水性消泡剂

礦物油混合物

可應用於水性塗料、水性油墨、水性接著劑。

ELIFOM-6632-消泡剂

含疏水粒子的聚矽氧烷

適用於高固型份、高膜厚環氧地板塗料及網印油墨的抑泡及消泡。

ELIFOM-6232-消泡剂

非矽酮的破泡高分子

應用於抑泡及消除木器底漆製造過程或添加填充料所導致的氣泡。

ELIFOM-6231-消泡剂

非矽酮的破泡高分子

應用於UV硬化塗料、不飽和聚酯塗料與環氧塗料的抑泡及脫泡。

ELIFOM-6107-消泡剂

改質聚矽氧烷

適用溶劑型、無溶劑型塗料的矽酮消泡劑及抑泡劑，在聚胺酯塗料厚膜的氣泡消除效果顯著。

ELIFOM-6106-消泡剂

改質聚矽氧烷

在聚胺酯塗料厚膜的氣泡消除效果顯著，用做木器塗料、塑膠塗料、NC噴漆等的消泡劑，相容性優秀，且很容易添加到系統中。

ELIFOM-6103-消泡剂

改質聚矽氧烷

用做木器塗料、塑膠塗料、NC噴漆等的消泡劑。同時在淋塗、滾塗的溶劑型塗料氣泡抑制與消除效果十分優秀。

ELIFOM-6101-消泡剂

改質聚矽氧烷

主要應用於塑膠漆、工業漆、木器漆等脫泡及消泡。

DMAST-2422-润湿分散剂

改質聚矽酮混合

用於溶劑型塗料的分散無機顏料，以防止顏料出現浮色和發花。特別適合於調和漆，也可於增加塗刷顏色均勻性 。

DMAST-2307-润湿分散剂

陰離子性高分子

具高性價比的溶劑型濕潤、分散劑，適用於碳黑等有機顏料的研磨分散。

DMAST-2306-润湿分散剂

含羧酸高分子

適用於無機顏料或有機無機顏料共研磨時之分散，並對二氧化鈦和其他顏料混用時之分色具抑制效果，也可以避免產生沉積效果。

DMAST-2306S-润湿分散剂

含羧酸高分子與改質聚矽酮的混合物

適用於無機顏料或有機無機顏料共研磨時之分散，並對二氧化鈦和其他顏料混用時之分色、浮色具抑制效果，也可以避免產生沉積效果。

DMAST-2303-润湿分散剂

電中性聚醯胺與聚酯混合物

特別適合於木器底漆的潤濕分散及膨潤土的濕。

SOPH-SAP-A2182-水性密著剂

SOPHI-SAP-A2182是水性磷酸酯密着剂，对于金属底材有十分优秀的密着性，广泛用于底漆的金属底材密着促进剂，增加对面漆的密着。

  • 产品性质
    1. 組成-------磷酸脂高分子
    2. 外觀-------微黃色液體
    3. 比重-------1.12-1.15 (@ 25℃)
    4. 不揮發份-------68.0-72.0%
    5. 溶剂-------DPM
    6. 酸价-------20.0~40.0 mg KOH/g
  • 特徵
    1. SOPHI SAP-A2182可以做為對水性金屬密著塗料的密著促進劑， 提供塗料對不銹鋼、金屬、玻璃基材的接著性。系統可為丙烯酸、雙組份聚氨酯及環氧。
    2. 適合室溫硬化或低溫化系統，也可以應用於水性烤漆。
    3. 適合水性系統，操作性方便。
    4. SOPHI SAP-A2182具有優良的塗膜的耐水性與耐鹽霧性。若與磷酸鋅併用，效果更顯著。
  • 应用
    1. 水性金屬塗料
    2. 水性烤漆
    3. 水性丙烯酸塗料
    4. 水性醇酸塗料
    5. 水性聚氨酯塗料
