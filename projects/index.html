SOPH-SAP-A2182-水性密著剂

SOPHI-SAP-A2182是水性磷酸酯密着剂，对于金属底材有十分优秀的密着性，广泛用于底漆的金属底材密着促进剂，增加对面漆的密着。

SOPH-SAP-A2182-水性密著剂

SOPHI-SAP-A2182是水性磷酸酯密着剂，对于金属底材有十分优秀的密着性，广泛用于底漆的金属底材密着促进剂，增加对面漆的密着。

  • 产品性质
    1. 組成-------磷酸脂高分子
    2. 外觀-------微黃色液體
    3. 比重-------1.12-1.15 (@ 25℃)
    4. 不揮發份-------68.0-72.0%
    5. 溶剂-------DPM
    6. 酸价-------20.0~40.0 mg KOH/g
  • 特徵
    1. SOPHI SAP-A2182可以做為對水性金屬密著塗料的密著促進劑， 提供塗料對不銹鋼、金屬、玻璃基材的接著性。系統可為丙烯酸、雙組份聚氨酯及環氧。
    2. 適合室溫硬化或低溫化系統，也可以應用於水性烤漆。
    3. 適合水性系統，操作性方便。
    4. SOPHI SAP-A2182具有優良的塗膜的耐水性與耐鹽霧性。若與磷酸鋅併用，效果更顯著。
  • 应用
    1. 水性金屬塗料
    2. 水性烤漆
    3. 水性丙烯酸塗料
    4. 水性醇酸塗料
    5. 水性聚氨酯塗料
